About the Game

"Theater of the Mind is a surreal visual novel set in the subconscious of a character with repressed trauma. Each act represents an emotion turned into a stage. Each scene, an unanswered question."

Currently in alpha development using Python with Pygame, the game explores psychological themes through interactive narrative and symbolic puzzles.

FEATURES:

  • Narrative divided into theatrical acts
  • Psychological puzzles
  • Dialogue choices that alter the story
  • Adaptive emotional soundtrack
  • Unique puppet theater visual style
